Note about "Pixel Pitch"

Please view this page with Firefox Dev Tools open. Note that in "mobile preview" mode, there is a menu option to change the "DPR" ("Device Pixel Ratio").

The DPR is the setting for high resolution screens, such as Apple "Retina" displays. Devices can have DPRs of anywhere from 1 to 4. For example, an iPhone 6/7/8 Plus has a DPR of 3.

This means your 200px image on a regular screen requires a 600px image to display properly on an iPhone 8+.

No CSS sizing, no constraints other than viewport size
